# Projeto prático

## Objetivo

Desenvolver um sistema web para gerenciamento de produtos.

### Requisitos do sistema:

- Recrusos a serem implementados:
    - Cadastrar.
    - Atualizar.
    - Pesquisar.
    - Remover.
- O layout do sistema deve ser responsívo.
